# Contributing to SynGP500

## Reporting Clinical Inaccuracies

If you identify clinical inaccuracies in specific cases, please email **piyawoot.song@gmail.com** with:
- File name and line number(s)
- Description of the inaccuracy
- Reference to guidelines/evidence (if applicable)

## Other Contributions

We welcome:
- Documentation improvements and clarifications
- Research findings using SynGP500 (please share your publications!)
- Suggestions for dataset enhancements

Email piyawoot.song@gmail.com with your ideas.

**Out of scope:** Requests for real patient data, clinical decision support features, or non-constructive criticism.
